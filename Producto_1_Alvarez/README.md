# Producto 1 - Ensamblador 8086

Este repositorio contiene los programas solicitados en el Producto 1, desarrollados en lenguaje ensamblador para el procesador Intel 8086.

## Alumno
**Nombre:** Leonel M. Alvarez Principe  
**Curso:** Arquitectura de Software  
**Universidad:** Universidad Continental  

## Contenido
| Ejercicio | Archivo | Descripción |
|----------|---------|-------------|
| 1 | ejercicio1_if.asm | Implementa condición IF-THEN-ELSE. |
| 2 | ejercicio2_while_fibonacci.asm | Calcula valores tipo Fibonacci con WHILE. |
| 3 | ejercicio3_mcd.asm | Máximo común divisor usando restas de Euclides. |
| 4 | ejercicio4_swap.asm | WHILE + condición final con intercambio (swap). |

## Cómo ejecutar
1. Abrir el archivo en **emu8086** o **MASM**.
2. Ensamblar y ejecutar paso a paso para observar los registros.
